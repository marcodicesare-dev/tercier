## Why These Two Paths

- **AI-Native Profitable Compounding**: assumes Tercier reaches early profitability, reinvests revenue, and uses AI leverage to operate with much higher output per person than classic SaaS.
- **VC-Accelerated Category Capture**: assumes capital mainly compresses time in integrations, enterprise credibility, implementation capacity, and international / group-level GTM.

This is intentionally not an old-school `money = 1:1 growth` model.
